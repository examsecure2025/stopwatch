# Stopwatch

## Description

A digital stopwatch application that allows you to track time with precision. The stopwatch displays time in minutes, seconds, and centiseconds format. It includes lap functionality to record multiple time intervals during a single session. The interface is clean and easy to use with smooth animations and responsive design.

## Features

- Start, pause, and reset functionality
- Lap time recording
- Time display in MM:SS:MS format
- Lap history list
- Loading animations for better user experience
- Responsive design for all devices

## How to Run

1. Open the 'stopwatch' folder
2. Double-click on 'index.html' file
3. The stopwatch will open in your default web browser
4. Click Start to begin timing

## Files

- index.html - Main HTML structure
- style.css - Styling and layout
- script.js - Stopwatch functionality

## Usage

- Click Start button to begin the stopwatch
- Click Pause button to pause the timer
- Click Lap button to record a lap time
- Click Reset button to clear all times and start over
- View all recorded laps in the laps list below
